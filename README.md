・ツールについて

当ツールはニコカラメーカーで「空行位置にページ区切りを設定」を使うための行区切りを自動で行う支援ツールです


・使い方

入力欄にRhythmicaLyricでタイムタグ設定済みの歌詞のエディタモードで表示されるテキストを貼り付けてください。

「自動整形」を押すと出力欄に2行ごとに空行で区切った歌詞が出力されます。

前の行の行末と次の行の行頭のタイムタグを比較して

時間差閾値(デフォルト210)に設定した値以上の差がある場合は1行でも空行を追加します。

閾値と改行間隔は上部で設定することで任意の値に変更できます。

アプリを終了したタイミングでJSONファイルに記憶されます。

初期値に戻す場合はデフォルトに戻すを押してください。
