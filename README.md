# 歌詞改行ツール

ニコカラ制作向けの歌詞整形支援ツールです。

RhythmicaLyricsで作成したタイムタグ付き歌詞を、ニコカラメーカーの
「空行位置にページ区切りを設定」機能で使用しやすい形式へ自動整形します。

---

## ダウンロード

最新版は **Releases** からダウンロードできます。

ZIPを解凍後、

```
歌詞改行ツール.exe
```

を実行してください。

---

## 主な機能

### 自動整形

タイムタグ付き歌詞を解析し、自動で空行を追加します。

* 基本設定では2行ごとに空行を追加
* 前後のタイムタグの時間差を比較し、必要な箇所には1行でも空行を追加
* 改行間隔・時間差閾値は任意に変更可能

時間差閾値は、ニコカラメーカーで設定している

* ワイプ前の表示時間
* ワイプ後の表示時間
* 歌詞の表示間隔

の合計値に合わせることを推奨します。

この設定にすることで、ニコカラメーカーの
「一定行数ごとにページ区切りを設定」と同等の動作になります。

---

## 使い方

1. RhythmicaLyricsでタイムタグ設定済みの歌詞を作成します。
2. エディタモードで表示されるテキストをコピーします。
3. 歌詞改行ツールの入力欄へ貼り付けます。
4. 「自動整形」を押します。
5. 出力欄に整形済み歌詞が表示されます。(同時にクリップボードにコピーされます)

---

## 設定について

画面上部から以下の設定を変更できます。

* 改行間隔
* 時間差閾値
* タイムタグ自動並び替えON/OFF
* 先頭タイムタグ違反の無視ON/OFF

設定値はアプリ終了時にJSONファイルへ保存され、次回起動時に復元されます。

---

# タイムタグ検査機能

タイムタグの巻き戻りを検知します

デフォルトで行頭の巻き戻り(前行の終わりより早い場合)は無視する設定になっています

## 機能

前後のタイムタグを比較し、

時間が巻き戻っている箇所(後ろのタイムタグが前より早い) を検出します。

該当箇所はハイライト表示され、検査画面に行番号を表示します。

また、設定から

```
各行最初のタイムタグ違反は無視する
```

のON/OFFを変更できます。

---

# タイムタグ並び替え機能

設定でON/OFFを変更できます。

通常使用ではONのままで問題ありません。

自動整形実行時に、各行の先頭タイムタグを基準に昇順で並び替えます。

以下のような場合に便利です。

* 複数の歌詞データを結合した場合
* 後からコーラスパートを追加した場合
* タイムタグ順が崩れた場合

## 注意事項

本ツールはRhythmicaLyricsで作成したタイムタグ歌詞を対象に設計しています。

その他の形式の歌詞データでは正常に動作しない場合があります。
